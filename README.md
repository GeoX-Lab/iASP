# iASP:Implicit Alignment Subspace Projection for Parameter Efficient Continual Learning of Vision-Language Models
📢 News
[2026.05]: Repo created. Code and pre-trained models are being cleaned up and will be released soon. Stay tuned!

✨ Highlights

[1]: We formulate alignment reuse as an implicit soft projection in the expert-induced alignment function space, exposing limitations of visual-similarity-based sharing.

[2]: We propose iASP, a two-stage framework that leverages early router convergence to automate subspace identification, redundancy truncation, and task-agnostic deployment, providing an end-to-end solution for efficient VLM adaptation.

[3]: On the MTIL benchmark, iASP achieves state-of-the-art accuracy while dramatically improving training efficiency, reducing the average number of trainable parameters by 42.7\% compared to current SOTA methods.
